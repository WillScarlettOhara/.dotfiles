# 🚀 Dotfiles & Linux Zero-Touch Provisioning

This public repository contains my Linux configuration files managed with `stow`, along with a custom automation engine (`bootstrap.sh`) designed to reinstall my entire environment from scratch on a fresh machine with a single command.

> [!WARNING]
> **Security**: No sensitive data, local IP addresses, or passwords are stored here. All private data, SSH keys, and heavy volumes (Virtual Machines, Firefox profiles, Network Mounts) are encrypted end-to-end and stored in OneDrive via **Restic** and **Bitwarden**.

---

## 🛠️ Restoration Prerequisites

Before running the script on a fresh installation, you must have **4 secure notes** saved in your Bitwarden vault:

1. **`SSH GitHub`**: Contains your private SSH key (Base64) in the note body, and the public key in a custom field named `PUBLIC_KEY`.
2. **`Config Rclone`**: Contains the OneDrive token generated by `rclone`.
3. **`Restic Password`**: Contains the password to decrypt your Restic backups.
4. **`Network Config`**: Contains the local IP addresses and mapping for your internal machines.

---

## ⚡ One-Click Installation (Bootstrap)

Open a terminal on your new machine (compatible with Arch Linux / CachyOS) and run:

```bash
curl -fsSL https://raw.githubusercontent.com/WillScarlettOhara/.dotfiles/master/bootstrap.sh | bash
```
